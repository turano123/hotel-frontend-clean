# HMS Backend (Express + MongoDB)

## Hızlı Başlangıç
1) Bağımlılıkları yükleyin:
```bash
cd backend
npm install
```
2) `.env` dosyasını `.env.example`'dan kopyalayıp doldurun.
3) Seed verileri yükleyin (master admin + örnek oteller):
```bash
npm run seed
```
4) Sunucuyu başlatın:
```bash
npm start
```
API: http://localhost:5000
